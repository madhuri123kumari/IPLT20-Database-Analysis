# IPLT20-Database-Analysis

 IPL Analysis Dashboard (2008 - 2025)
This repository contains a comprehensive Power BI Data Analysis Project tracking Indian Premier League (IPL) performance history, with a deep-dive focus on individual seasons like 2024.

🚀 Key Concepts & Metrics
1)KPI Tracking: Real-time extraction of aggregate tournament stats including Total Matches (71), Total Sixes (1261), Total Fours (2175), Centuries (14), and Half-Centuries (128).
2)Dynamic Standings: A dynamic league Points Table displaying matches played (Pld), total wins, losses, and team logos.
3)Individual Accolades: High-impact visual cards highlighting the Orange Cap (Most Runs - V Kohli) and Purple Cap (Most Wickets - HV Patel) winners.
4)Player Performance Segments: Granular insights showcasing top individual boundary hitters, featuring players like TM Head (Total Fours) and Abhishek Sharma (Top Six Count).

🛠️ Step-by-Step Implementation
i)Data Acquisition & Cleaning: Imported multi-year IPL datasets spanning from 2008 to 2025. Cleaned structural inconsistencies, handled missing match results, and standardized team names using Power Query.
ii)Data Modeling: Established relationships between historical match schedules, individual player statistics, and team standings to build a cohesive star schema.
iii)DAX Calculations: Developed custom DAX measures to calculate complex aggregates, including running boundary counts, total tournament points, and dynamic rank headers.
iv)UI/UX Dashboard Design: Designed a professional, interactive interface in Power BI Desktop featuring custom-curated player image cards, custom themes, and an interactive season slicer for smooth user filtering.

🛠️ Core Technologies & Tools Used
